## 1. Цель и задачи работы

### Цель
Освоить методологию **модульного тестирования (Unit Testing)** в Python с использованием стандартного модуля `unittest`. Научиться применять тесты для проверки корректности функций и обработки исключительных ситуаций.

### Задачи
1.  Разработать систему расчета оценки студента.
2.  Реализовать набор Unit-тестов для всех функций.
3.  Обеспечить проверку всех веток кода, включая обработку ошибок.
4.  Оформить проект, включая файл `README.md` с результатами.

---

## 2. Описание решения

Проект состоит из двух основных файлов: `main.py` (основная логика) и `test_main.py` (тесты).

### `main.py`
Файл содержит три функции:
1.  **`calculate_average(grades)`**: Вычисляет средний балл. Содержит проверки на пустой список (`ValueError`) и некорректный тип данных (`TypeError`).
2.  **`determine_grade_letter(avg)`**: Определяет буквенную оценку (A, B, C, D, F) на основе среднего балла.
3.  **`student_report(name, grades)`**: Интеграционная функция, которая объединяет расчеты и формирует итоговый отчет.

### `test_main.py`
Файл содержит класс `TestGrades` с **пятью тестами**, которые проверяют:
* Корректность расчета среднего балла.
* Обработку исключения `ValueError` (при пустом вводе).
* Обработку исключения `TypeError` (при нечисловом вводе).
* Правильность присвоения всех буквенных оценок.
* Корректность итогового отчета (`student_report`).
